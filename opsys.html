Operating Systems Terminology
Term Definition
operating system The software that controls the overall operation of a computer.
job queue A storage organization in mass storage waiting for execution.
FIFO Jobs in mass storage waiting for execution first in, first out.
interactive processing Allows a program being executed to carry on a dialogue with the user.
time-sharing A program that provides service to multiple users at the same time.
multiprogramming Time is divided into intervals and then the execution of each job is restricted to only one interval at a time.
application software Consists of the programs performing tasks particular to the machine's utilization.
system software Provides the infrastructure that the application software requires.
utility software Software units collectively in the class of system software.
graphical user interface (GUI) Objects to be manipulated are represented pictorially on the display as icons.
shell Older user interfaces.
kernel The internal part of an operating system.
device driver The software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
file manager Whose job is to coordinate the user of the machine's mass storage facilities.
memory manager Coordinating the machine's use of main memory.
virtual memory A large fictional memory space created by paging.
boot strapping Transfers the operating system from mass storage into main memory.
ROM Memory that can be read but not altered.
deadlock The condition in which two or more processes are blocked from programming because each is waiting for a resource that is allocated to another.
spooling A technique that holds data for output at a later but more convenient time.
super user (or administrator) Accounts that are established by a person.
multi-factor authentication A secure way to access computer accounts using two password factors.
